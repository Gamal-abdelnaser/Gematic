# Gamal Abdelnaser – Frontend Portfolio

## Overview

This repository contains the source code of my personal **frontend portfolio website**.  
The website is designed to showcase my projects, skills, and experience as a frontend developer, with a strong focus on clean UI, smooth interactions, and modern web standards.

The portfolio highlights real-world projects in fitness platforms, dashboards, calculators, and responsive web applications.

---

## Features

- Modern and clean UI
- Fully responsive design (mobile-first)
- Smooth animations and transitions
- Projects showcase with live demos and source code links
- Optimized performance and accessibility

---

## Tech Stack

- **React**
- **Vite**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**
- **Framer Motion**

---

## Project Structure

The project follows a scalable and maintainable structure using reusable components and modern React best practices.
